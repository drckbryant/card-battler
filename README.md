# Card Battler Prototype

This is a browser-based card battler built using React, TypeScript, and Vite.

## Overview
This project is a personal development project focused on exploring systems design, front-end development, and interactive application logic.

The goal is to experiment with:
- Modular combat systems
- Card-based mechanics
- State management and game flow
- Scalable architecture for interactive applications

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Features (Work in Progress)
- Turn-based combat system
- Card-driven actions and effects
- Modular game logic structure
- UI components for gameplay interaction

## Running the Project

Clone the repository and install dependencies:

```bash
npm install
npm run dev
```

## Notes
This project is an ongoing work in progress and is used as a learning and experimentation platform for building interactive web applications.
